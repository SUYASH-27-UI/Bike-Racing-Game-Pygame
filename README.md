# Bike Racing Game (Python Pygame)

A simple 2D bike racing game developed using Python and Pygame. The player controls a bike, avoids incoming enemy vehicles, and tries to achieve the highest possible score. The game also includes background music, crash sound effects, and a custom road background.

## Features

- 2D Bike Racing Gameplay
- Keyboard Controls (Left & Right Arrow Keys)
- Moving Enemy Vehicle
- Score Counter
- Collision Detection
- Road Background Image
- Engine Sound Effect
- Crash Sound Effect
- Smooth Gameplay using Pygame

## Technologies Used

- Python
- Pygame

## Assets

- `road.png` – Road background image
- `engine.wav` – Engine sound effect
- `crash.wav` – Crash sound effect

## Controls

- ⬅ Left Arrow – Move Left
- ➡ Right Arrow – Move Right

## How to Run

1. Install Python.
2. Install Pygame:

```bash
pip install pygame
```

3. Make sure these files are in the same folder:

- `racing_game.py`
- `road.png`
- `engine.wav`
- `crash.wav`

4. Run the game:

```bash
python racing_game.py
```
